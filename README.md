# meu-portifolio

Portfólio estático inspirado em um tema “Van Gogh / Starry Night”.

## Como abrir

- Abra o arquivo `index.html` no navegador (duplo clique).

## O que você deve trocar primeiro

- `index.html`
  - Nome e textos (Hero + Sobre)
  - Projetos (título, descrição, tags e links de Demo/Código)
  - Contato (email, LinkedIn, GitHub)
- `script.js`
  - Troque `seuemail@exemplo.com` pelo seu email real (envio do formulário)

## Estrutura

- `index.html`: conteúdo
- `styles.css`: tema/estilo (Van Gogh)
- `script.js`: menu mobile, parallax leve e formulário via `mailto:`
